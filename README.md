# vyper
## Installation of Vyper 

    PIP - https://vyper.readthedocs.io/en/stable/installing-vyper.html

## Create virtual env 
``` bash
$ virtualenv -p python3 venv
$ source venv/bin/activate
```
## Install Vyper 

``` bash
$pip3 install vyper
```
## Now create file to write program in vyper with (.vy) extension.
```bash
$ touch hello.vy
```
## Compile program 
``` bash
$vyper hello.vy
```
